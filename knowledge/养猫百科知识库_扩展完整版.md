## 一、原文保留 + 补充说明（示范）

以下演示「经验笔记 + 百科补充」的写法：**先保留第一人称/群友原文，再跟【补充知识】。**

### 1.1 猫藓治疗（原文摘录）

**原文：**

当时给刘福团治猫藓，外用的盐酸特比萘芬喷剂，这个药不能和碘伏一起用。内服药我没给他吃那个伊曲康唑，因为我做过攻略的时候说有点伤肝，他太小了我就没敢给他吃。然后买了那种营养补剂，一顿饭里滴两泵。她住的常去的地方一天用次氯酸溶液消杀两次，包括床底。我天天拖干净了再喷这个溶液。然后多晒太阳。

**【补充知识】**

猫藓（真菌感染）是猫咪非常常见的皮肤病之一，尤其在以下情况下更容易发生：

- 幼猫
- 抵抗力较低
- 环境潮湿
- 多猫家庭

**常见症状包括：**

- 局部掉毛
- 皮肤结痂
- 圆形脱毛斑
- 皮肤发红

**常见治疗方案包括：**

- **外用药：** 盐酸特比萘芬、咪康唑、酮康唑等（具体与疗程须兽医指导）
- **碘伏：** 可与部分方案配合讨论，但**不宜与特比萘芬等外用方案随意混用或同区反复叠涂**——以药品说明与医嘱为准
- **口服药：** 伊曲康唑、灰黄霉素等，**肝肾功能与年龄需兽医评估**；幼猫一般优先「外用药 + 环境管理」

**环境消毒常见方式：** 次氯酸（按说明稀释与通风）、宠物专用消毒喷雾、紫外线杀菌灯（注意人与宠物安全）

**同时保持：** 多晒太阳、减少潮湿、定期清洁猫窝与织物

**这样做的好处：** 原文保留真实决策过程；补充段落在网站上可单独作为「百科摘要」区块，利于 SEO 与 RAG 切片。

## 四、猫咪疾病库（目标约 50 条）

> 文档若仅详细写过猫藓，库内仍应按「病种」独立成条，便于网站 `/diseases` 或 `?hub=disease` 检索。

### 4.1 猫藓（与第一节呼应）

- **概要：** 皮肤真菌感染，常见犬小孢子菌等
- **就医：** 确诊与治疗方式以兽医为准；幼猫优先外用药 + 环境管理

### 4.2 猫瘟（猫泛白细胞减少症）（示例扩展）

**症状：** 发烧、呕吐、腹泻、精神差、白细胞下降（以检验为准）

**传播：** 粪便、分泌物、污染环境与器具；接触传播

**预防：** 核心免疫（如「猫三联」等）按程序完成；新猫隔离；环境消毒

**说明：** 急性、传染性强，**疑似即就医**，勿自行按网络方案治疗。

### 4.3 猫口炎（示例扩展）

**症状：** 口臭、流口水、进食困难、疼痛表现

**治疗思路：** 消炎、控制感染、疼痛管理；顽固病例可能需拔牙等手术方案——**全部由兽医评估**

## 五、猫咪行为问题（目标约 30 条）

> 网站高需求模块：用户常搜「为什么」，适合 FAQ + 长文。

### 5.1 猫为什么半夜跑酷

**可能原因：** 精力过剩；晨昏活动习性；白天睡眠过多；环境刺激不足

**可尝试：** 傍晚固定陪玩约 20 分钟；睡前少量进食（与肥胖管理平衡）；猫爬架、隧道等丰容；保持白天也有一定活动

**就医提示：** 若伴随嚎叫、排尿异常、老年猫突然行为巨变，先排除疼痛与内科问题。

### 5.2 猫为什么踩奶

**原因：** 幼年期哺乳行为的残留；放松、舒适；对环境与主人信任的表现

**是否异常：** 多数情况下属正常行为，无需纠正。

## 六、猫咪用品库（目标约 150 条）

> 可做独立「用品数据库」，支撑推荐位与变现；字段见本文第八节。

### 6.1 梳毛用品（示例）

**短毛猫：** Kingpet 针梳、喵梵思排梳、宠物硅胶梳等（品牌仅作检索关键词示例）

**长毛猫：** 船记 DL103、金属针梳、排梳、修眉刀（谨慎处理打结，避免贴皮剪切）

**补充：** 长毛建议每天梳毛，减少毛球与打结；换毛季可增加频次。

### 6.2 猫砂（示例）

**膨润土猫砂**

- **优点：** 结团快、除臭相对直观、入门价位常见
- **缺点：** 粉尘相对多，敏感呼吸道需注意

**推荐品牌（示例关键词）：** pidan、N1、Ever Clean 等——以个体猫偏好与家庭清洁成本为准。

### 6.3 猫粮类型与营养结构（内容向）

**常见形态：** 干粮、冻干、主食罐、鲜食/熟自制（熟自制需配方与营养平衡，不建议盲目跟风）

**营养结构（常识向，非处方）：** 猫为专性肉食动物，商业主食应满足相应标准（如 AAFCO / FEDIAF 等）；讨论「高蛋白」时须结合肾健康与个体疾病，**肾病猫饮食遵医嘱**。

**经验性区间（仅作内容库表述，非喂养标准）：** 粗蛋白、脂肪比例常见讨论区间可写在「百科说明」中，**具体选粮以包装与兽医为准**。

## 八、网站数据库结构（建议）

以下与 Next.js / Notion / 任意后端均可映射；字段名可英文化后入库。

### 表 1：问题库 `questions`

| 字段 | 说明 |
|------|------|
| question_id | 主键 |
| title | 问题标题（贴近搜索句） |
| category | 分类 |
| answer | 正文（可 Markdown） |
| related_products | 关联用品 ID 列表或名称列表 |
| seo_title | SEO 标题 |
| seo_description | SEO 描述（摘要） |

### 表 2：用品库 `products`

| 字段 | 说明 |
|------|------|
| product_id | 主键 |
| product_name | 名称 |
| category | 分类 |
| price_range | 价格区间 |
| description | 描述 |
| recommend_reason | 推荐理由 |

### 表 3：疾病库 `diseases`

| 字段 | 说明 |
|------|------|
| disease_id | 主键 |
| name | 疾病名 |
| symptoms | 症状 |
| cause | 原因 |
| treatment | 治疗（科普向，强调遵医嘱） |
| prevention | 预防 |

**可选扩展表：** `behaviors`（行为问题）、`guides`（长文指南）、`seo_pages`（专题落地页）。
