# Cat Knowledge Base — 信息架构与页面说明

本文档整合产品侧 IA 要求，并标注与本地资料 **`新手养猫清单.docx`**、**`猫咪指南.pdf`**（Downloads）对应的落点：详细摘录在 `src/data/guides.json`（由 `guides.csv` 同步）及问答库中。

## 核心原则：以「用户搜索问题」为主键

- 知识组织**不要**只按「疾病 / 行为 / 饮食 / 护理」切块让用户找分类。
- **要**按真实搜索句组织，例如：猫咪为什么吐、为什么不吃饭、为什么一直叫、为什么咬人。
- 分类（category）仅作筛选与运营标签；**检索与首页入口优先用自然语言问题**。

## 一、首页（Home）

首页只做三件事：

1. **搜索框（最重要）**  
   示例占位见 `src/data/home-content.json` → `search_examples`。

2. **常见问题入口 — 8 张大卡片**  
   同上文件 → `category_cards`（新手 / 吃什么 / 健康疾病 / 绝育 / 行为 / 清洁护理 / 用品 / 紧急情况）。

3. **热门问题**  
   `hot_question_titles` 与 `getPopularQuestions()` 对齐；条目需在 `questions.csv` 中有同标题行。

## 二、养猫问题库（最核心）

- 列表 → 点击进入详情。
- 建议每条结构化（可在 Notion/ CMS 中扩展字段，当前 JSON 用 `description` + `causes` + `solutions` 承载）：
  - **问题**
  - **简短结论** → 对应 `description` 首段或全文首句
  - **详细解释** → `description` 展开
  - **需要注意** → `solutions` 中风险提示与就医边界
  - **推荐用品** → `recommended_products`

**示例（公猫绝育与情绪）**：库内已有「公猫绝育会恨主人吗」等绝育专题问答。

## 三、猫咪健康库

高搜索量模块；站点内对应 **`/questions?hub=disease`**（肠胃 / 健康 / 护理分类下的问答）。

每条疾病向内容建议包含：

- 症状、原因、是否严重、什么时候去医院、治疗方式（与兽医一致）。

常见主题示例：猫癣、猫鼻支、腹泻、呕吐、尿闭、耳螨等 — 见问答与指南长文。

## 四、猫咪用品库

- 路由：`/products`；未来变现位。
- 分类维度示例：猫粮、猫砂、猫砂盆、猫抓板、玩具、猫窝、营养品、清洁用品（与 `Product` 类型及运营配置对齐）。

## 五、猫咪行为问题

- 路由：`/questions?hub=behavior`。
- 结构建议：为什么会这样、常见原因、解决办法、需要多久改善。

## 六、（可选）新手养猫指南

- 路由：`/guide`。
- 流量入口示例：第一次养猫准备什么、一个月花费、疫苗时间表、绝育指南等 — **`guides.csv` / `新手养猫清单` 摘录**已覆盖部分。

## 七、极简导航（5 + 1）

| 菜单       | 路径 |
|------------|------|
| 养猫问题   | `/questions` |
| 猫咪健康   | `/questions?hub=disease` |
| 猫咪行为   | `/questions?hub=behavior` |
| 猫咪用品   | `/products` |
| 新手指南   | `/guide` |
| AI 问答（建议） | `/ai` — 基于 `answerByKnowledgeBase` 的 RAG 式检索回答 |

「选购专区」`/zones` 保留为二级入口（页脚或用品页内链），不占用五个主菜单名额。

## 八、AI 产品与 RAG

- 用户输入自然语言问题 → 在知识库中检索相关问答与用品 → 拼接摘要回答。
- 实现入口：`/ai`；逻辑：`src/lib/data.ts` → `answerByKnowledgeBase`。

## 九、猫咪绝育专题（公猫 vs 母猫）

- **长文对照**：指南库条目《公猫与母猫绝育：差异、护理要点与常见误区》含手术方式、术前术后、对照表与 5 条易忽略点。  
- **FAQ Top 20**：问答库已收录价格、性格、乱尿、发情期手术、禁食禁水、拆线、洗澡、寿命等条目；另见指南《猫咪绝育 · FAQ Top 20 索引与知识库分组》用于 **Notion/网站分组**（绝育价格｜年龄｜术前｜术中｜术后｜行为变化）。  
- **免责**：禁食禁水时长、麻醉方式与用药以医院书面医嘱为准。

## 十、资料来源声明

- **`新手养猫清单.docx`**：用品清单、品牌功课、驱虫与常备药思路等 — 已摘要进指南与用品描述，非完整转载。
- **`猫咪指南.pdf`**：奶猫保暖与喂食频次、去尿渍配方、梳毛工具、猫粮罐头冻干品牌讨论、绝育套餐参考、护理补剂等 — 与问答条目交叉引用。

涉及用药、麻醉、剂量、绝育与驱虫等，**一律以执业兽医为准**。
