# 养猫问题库（可读版）

总条数：326

## 肠胃问题（67）

### 1. 猫拉稀怎么办
- ID：`q1`
- 描述：软便或水样便先排查饮食与应激。
- 原因：突然换粮、应激、零食过多
- 方案：停止零食；喂益生菌；观察24小时精神状态
- 推荐用品：宠物益生菌、低敏主粮

### 2. 猫呕吐
- ID：`q2`
- 描述：偶发吐粮常见，频繁呕吐需重视。
- 原因：吃太快、毛球刺激、换粮过快
- 方案：少量多餐；使用慢食碗；频繁呕吐及时就医
- 推荐用品：慢食碗、化毛膏

### 3. 猫不吃猫粮
- ID：`q3`
- 描述：拒食可能与挑食或口腔不适相关。
- 原因：挑食、环境变化、口腔问题
- 方案：定时定量；减少零食；持续拒食就医
- 推荐用品：适口性猫粮、主食罐

### 4. 猫便秘怎么办
- ID：`q4`
- 描述：排便间隔过长需补水与饮食调整。
- 原因：饮水少、纤维不足、运动少
- 方案：增加饮水；湿粮辅助；必要时就医
- 推荐用品：流动饮水机、湿粮

### 5. 换粮后软便
- ID：`q5`
- 描述：换粮过快容易短期软便。
- 原因：过渡期太短、新粮不耐受、喂食过量
- 方案：7天渐进换粮；控制喂食量；必要时回退旧粮
- 推荐用品：宠物益生菌、低敏猫粮

### 6. 奶猫喂养频次怎么安排
- ID：`q23`
- 描述：奶猫喂养需高频少量，避免呛奶和过量。
- 原因：喂食间隔不合理、奶量控制不当、喂奶姿势错误
- 方案：0-1周约每2小时喂一次；2-3周每3-4小时一次；使用专用奶瓶俯卧喂养
- 推荐用品：猫用奶瓶、猫奶粉

### 7. 幼猫断奶怎么过渡
- ID：`q25`
- 描述：断奶阶段需要循序渐进，避免肠胃负担过重。
- 原因：直接更换固体粮、进食过快、适口性差
- 方案：3-4周开始奶泡幼猫粮；逐步减少奶量；观察便便和体重变化
- 推荐用品：幼猫粮、主食罐

### 1. 猫黑下巴
- ID：`q7`
- 描述：下巴黑点常见于皮脂堆积。
- 原因：油脂分泌、塑料碗刺激、清洁不足
- 方案：更换陶瓷或玻璃等易清洁食碗；每日清洁下巴；反复发作就医
- 推荐用品：陶瓷平口猫碗、宠物湿巾

### 2. 猫泪痕
- ID：`q8`
- 描述：轻度泪痕可通过清洁与饮食改善。
- 原因：眼部刺激、饮食问题、泪道堵塞
- 方案：每天清洁眼周；调整饮食；持续加重就医
- 推荐用品：宠物眼部湿巾、低敏猫粮

### 3. 猫掉毛严重
- ID：`q9`
- 描述：换毛季掉毛增多，需规律护理。

- 方案：每天梳毛；补充鱼油；皮肤异常就医
- 推荐用品：针梳、鱼油

### 4. 猫耳朵脏

- 推荐用品：洗耳液、棉片

## 健康问题（99）

### 1. 猫口臭
- ID：`q6`
- 描述：持续口臭可能提示口腔或肠胃问题。
- 原因：牙结石、牙龈炎、口腔清洁不足
- 方案：定期刷牙；使用洁牙零食；严重时口腔检查
- 推荐用品：猫咪牙刷、洁牙凝胶

### 2. 猫乱尿
- ID：`q12`
- 描述：乱尿可能是行为问题也可能泌尿问题。
- 原因：猫砂盆不洁、应激、泌尿不适
- 方案：增加猫砂盆；保持清洁；频繁蹲盆尽快就医
- 推荐用品：豆腐猫砂、猫砂盆

### 3. 猫咪应激反应
- ID：`q16`
- 描述：搬家或新成员可能引发应激。
- 原因：环境变化、外出就医、陌生气味
- 方案：提供安静空间；保持作息；持续不适就医
- 推荐用品：费洛蒙扩散器、舒缓零食

### 4. 猫打喷嚏
- ID：`q17`
- 描述：偶发喷嚏常见，伴随症状需关注。
- 原因：灰尘刺激、温差变化、上呼吸道感染
- 方案：保持通风；减少粉尘；流鼻涕发热就医
- 推荐用品：低尘猫砂、空气净化器

### 5. 猫频繁进出猫砂盆
- ID：`q19`
- 描述：疑似泌尿不适要尽快处理。
- 原因：泌尿刺激、饮水不足、应激
- 方案：观察尿量；增加饮水点；异常立即就医
- 推荐用品：饮水机、泌尿护理粮

### 6. 猫咪食欲下降
- ID：`q20`
- 描述：短期食欲下降需排查环境和疾病。
- 原因：天气变化、应激、口腔或肠胃不适
- 方案：提供新鲜食物；观察精神状态；超过24小时就医
- 推荐用品：主食罐、营养膏

### 7. 猫癣怎么护理
- ID：`q21`
- 描述：猫癣常见于免疫力低或潮湿环境，需要治疗与环境管理同步。
- 原因：真菌感染、免疫力下降、环境潮湿
- 方案：按医嘱外用药；隔离并清洁常活动区；定期消杀并多晒太阳
- 推荐用品：抗真菌喷剂、次氯酸消毒液
